Display voting buttons using emotions on any page, post or product. Buttons are rendered using the [buttons_like_computy] shortcode.

== Description ==
A distinctive feature of the counter from other similar ones is the blocking of cheat likes, as unique visitors are recorded through sessions.

== Testing ==
You can test the plugin on [**this page**](https://demo.tastewp.com/like-computy)

== Installation ==

1. Upload dir `like-computy` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
